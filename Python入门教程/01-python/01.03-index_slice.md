# 第 三 章 索引 和 切片

# 3.1 索引

## 3.1.1 正向
索引从```0```开始
```
s = 'hello'
s[0] # 'h'
```
## 3.1.2 反向
```
s = 'hello'
s[-1] # 'o'
s[-2] # 'l'
```
## 3.1.2 超出范围会报错
```
s = 'hello'
s[5] # IndexError: string index out of range
s[-6] # IndexError: string index out of range
```

# 3.2 分片 slice
var[lower:upper:step]
```
s = 'hello'
s[1:4]   # 'ell'
s[1:4:2] # 'el'
s[:4]    # 'hell'
s[1:]    # 'ello'
s[:]     # 'hello'
s[4:1:-1]# 'oll'
```
