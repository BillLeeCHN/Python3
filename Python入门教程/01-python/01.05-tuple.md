# 第 五 章 元组



# 5.1 操作

与列表相似，元组Tuple也是个有序序列，但是元组是**不可变**的，用```()```生成。
```
t = (10, 11, 12, 13, 14) # method 1
t = 10, 11, 12, 13, 14   # method 2
t[1] # 11 
t[1:4] # (11,12,13)
t[0] = 1 # TypeError: 'tuple' object does not support item assignment
```
# 5.2 与```list```转换
```
a = [1,2,3]
b = tuple(a) # 转换成tuple
b # (1,2,3)
c = list(b) # 转换成list
c # [1,2,3]
```
# 5.3 count
```
t = (1,2,2,4,2)
t.count(2) # 3
t.index(2) # 1
```
